# Индексация публичных маршрутов — план реализации

> **Для агентного выполнения:** выполнять задачи последовательно в текущей сессии; новые автотесты не добавлять без отдельного разрешения пользователя.

**Цель:** обеспечить известным маршрутам `/maps/<slug>` успешный HTTP-ответ после публикации на GitHub Pages, добавить базовые SEO-метаданные и опубликовать `robots.txt` с `sitemap.xml`.

**Архитектура:** Vite-плагин после сборки читает итоговый `index.html`, создаёт статический `maps/<slug>/index.html` для каждой записи `mapCatalog`, подставляет уникальные SEO-метаданные и генерирует служебные файлы поисковых роботов. `404.html` сохраняется для неизвестных маршрутов, а React Router и публичные URL не меняются.

**Стек:** React 19, Vite 8, TypeScript, GitHub Pages.
